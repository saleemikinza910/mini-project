 ✨ Overview

This project is a **high-end luxury jewelry website UI clone inspired by Tiffany & Co.**

It focuses on:
- Premium visual design 🎨  
- Luxury brand experience 💎  
- Smooth interactions ⚡  
- Real-world e-commerce layout 🛍️  

Built using only **HTML, CSS, and JavaScript (no frameworks)**.

---
